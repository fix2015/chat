# Simple chat on reactJS and sockeIO

# Features
* reactjs
* socket io
* nodeJS
* google map

## 1. Setup
```bash
npm install
```
- install all npm and bower dependencies

## 2. Build production version
```bash
	npm run build
```
## 4. Start webserver
```bash
npm start
```

 `localhost:5000`
